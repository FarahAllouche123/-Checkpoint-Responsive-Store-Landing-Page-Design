Choix de conception

Le nom MaChic vient de la combinaison de “Magic” + “Chic”, reflétant l’élégance et le côté magique de la marque.
Les couleurs violettes et fuchsia ont été choisies pour représenter le raffinement et le côté féminin.
La mise en page utilise un carousel pour le hero et une grille CSS pour les produits, offrant une présentation claire et attractive.
Le design est entièrement responsive, avec des cartes produits interactives et des boutons visibles pour améliorer l’expérience utilisateur.
